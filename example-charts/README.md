# Example Charts

This folder contains production-ready example charts for the gallery feature.

## Quick Start

1. **Create a chart** in the chart editor
2. **Save as JSON** using the "Save Chart" button
3. **Rename** following the naming convention: `descriptive-name.json`
4. **Create SVG thumbnail** (same name as JSON)
5. **Move to appropriate folder** (e.g., `/bar-horizontal/`)

## Folder Structure

```
example-charts/
├── bar-horizontal/          # Horizontal bar charts
├── bar-vertical/            # Vertical bar charts
├── bar-grouped-horizontal/  # Grouped horizontal bars
├── bar-grouped-vertical/    # Grouped vertical bars
├── bar-stacked-horizontal/  # Stacked horizontal bars
├── bar-stacked-vertical/    # Stacked vertical bars
├── line/                    # Line charts
├── area/                    # Area charts
├── area-stacked/            # Stacked area charts
├── slope/                   # Slope charts
└── funnel/                  # Funnel charts
```

## File Naming Convention

- **Format**: `kebab-case-name.json` + `kebab-case-name.svg`
- **Example**: `modern-sales-dashboard.json` + `modern-sales-dashboard.svg`

## Requirements

Each example chart needs:
- ✅ JSON file with chart data and styles
- ✅ SVG thumbnail (800×600px recommended)
- ✅ Metadata in JSON (title, description, tags)
- ✅ Professional styling
- ✅ Tested in light and dark mode
